# 🏠 Homeplus

Homeplusは、生活をより便利にするためのホームダッシュボードアプリです。

## 開発環境

- Python
- Streamlit
- SQLite

## 起動方法

### ショートカット

WindowsのHomeplusショートカットをダブルクリック

### ターミナル

```bash
bash start_homeplus.sh
```

## ディレクトリ構成

```text
Home-plus/
├── app.py
├── assets/
├── data/
├── docs/
├── pages/
├── services/
├── tests/
└── start_homeplus.sh
```
